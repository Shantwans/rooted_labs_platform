This repository defines the system architecture, learning logic, and screen flows for the Rooted Labs AI literacy platform. It is a specification repo, not an implementation repo

# Rooted Labs Platform

Rooted Labs is an AI literacy learning platform designed for children, teachers, parents, and schools.

The platform delivers AI concepts through:
- Stories
- Visual explanations
- Interactive games
- Unplugged activities
- Guided reflection

The system is inspired by code.org-style progression:
- Level-based
- Locked flow
- Facilitator-gated learning

This repository defines:
- Learning architecture
- Level logic
- Role responsibilities
- Screen flows
- Future AI extensibility

⚠️ This is a specification repository.
Implementation should strictly follow the docs folder.
